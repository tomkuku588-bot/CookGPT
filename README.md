# CookGPT Legal Pages

Static legal page for CookGPT HarmonyOS listing review.

## Files

- `index.html`: one-page bilingual Privacy Policy and User Agreement.
- `.nojekyll`: keeps GitHub Pages from applying Jekyll processing.

## GitHub Pages

Use GitHub Pages with source set to the `main` branch root. The public URL should be:

`https://tomkuku588-bot.github.io/CookGPT/`

Use this URL as the privacy statement / user agreement URL when submitting CookGPT to AppGallery Connect.
