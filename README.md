# CookGPT Legal Pages

CookGPT 的 GitHub Pages 静态协议页面，用于 HarmonyOS 应用上架和审核。

## 推荐入口

- `index.html`: 中英文单页协议中心，包含隐私政策和用户协议。
- `privacy-policy.md`: 隐私政策 Markdown 版本。
- `user-agreement.md`: 用户协议 Markdown 版本。
- `privacy-and-user-agreement.md`: 隐私政策与用户协议合并 Markdown 版本。

## 兼容说明

- `.nojekyll` 保留，用于避免 GitHub Pages 对静态文件做 Jekyll 处理。
- 当前公开入口保持为 `https://tomkuku588-bot.github.io/CookGPT/`。
- 后续新增文件优先使用小写短横线命名。

## 部署说明

GitHub Pages 建议使用仓库发布分支的根目录：

1. 打开仓库 `Settings` > `Pages`。
2. `Build and deployment` 选择 `Deploy from a branch`。
3. 分支选择当前发布分支，目录选择 `/ (root)`。
4. 保存后等待 Pages 部署完成。
